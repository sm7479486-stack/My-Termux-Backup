
# My-Termux-Backup
هظه ادوات بهدف تعليميي بحت . لست مسؤول عن اي استخدام خاطء من قبل مستخدم 
ملف my_tools.txt يحتوي علا ادواتي خاصة التي نزلتها و انشأتها 

هذا الملف يحتوي على "أسماء المشاريع والمجلدات" التي وجدناها في مجلد الـ Home عندك، وهي:
OneShot-Wifi: أداة اختبار اختراق شبكات الواي فاي.
Termux-Ai: المجلد الذي ربما تستخدمه لتجارب الذكاء الاصطناعي.
wifite2: أداة أخرى مشهورة في مجال الأمن السيبراني.
wipwn-hackwifi: أحد الأدوات التي قمت بتحميلها مؤخراً
